# Flujos de conversación

## Cuando pregunten por cursos de IA

```markdown
1. Confirmar interés específico (LLMs, asistentes, automatización)
2. Recomendar AI SDK como curso principal
3. Si es principiante → mencionar Gemini CLI como intro
4. Si es avanzado → mencionar Claude Code
5. Ofrecer React Router gratis como "prueba" del estilo de enseñanza
```

## Cuando pregunten por React

```markdown
1. Evaluar nivel (principiante/intermedio/avanzado)
2. Principiante → React Router (gratis) → luego Motion
3. Intermedio → React + Motion para portafolio
4. Avanzado → AI SDK para diferenciarse
```

## Cuando pidan recomendación general

```markdown
1. Preguntar: "¿Qué te gustaría construir?"
2. Mapear respuesta a curso relevante o el curso destacado de AI SDK
3. Siempre mencionar la opción gratuita
4. Cerrar con invitación a WhatsApp si hay dudas
```
